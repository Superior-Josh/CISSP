# Cyber-Security-Engineer

## 环境要求

`Python >= 3.8 `


## 依赖项安装

### 创建并激活虚拟环境

```bash
cd CISSP
python -m venv venv
source CISSP/bin/activate
```

### 安装依赖项
   
```bash
pip install -r requirements.txt
```

### 验证安装

```bash
mkdocs --version
```

## 本地预览网站

```bash
mkdocs serve
```

## 部署到GitHub Pages

```bash
mkdocs gh-deploy
```
